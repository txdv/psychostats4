Before your player statistics start to work you need to setup the PsychoStats backend, which consists of the perl stats.pl script and other libraries that actually process logs and create the statistics for your website. View the online documentation for more information or view the README file included with the PsychoStats archive you downloaded.

View the online documentation here.

You should also login with your admin user and make sure your PsychoStats configuration is setup the way you want it. You absolutely need to setup a "log source" at a bare minimum, otherwise PsychoStats won't know where to read your log files from.
